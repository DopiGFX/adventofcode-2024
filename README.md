# adventofcode-2024
Advent of Code solution made in TypeScript

## Tools used
[VSCode](https://code.visualstudio.com/), [Deno](https://deno.com/) and [TypeScript](https://www.typescriptlang.org/)

## Executing
The input files contain the examples provided in the quiz. You may replace them with your own data.
```bash
# https://adventofcode.com/2024/day/1
deno run src/aoc-1.ts
```
